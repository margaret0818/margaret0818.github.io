Romanian Holidays


What is it?

My friend and I always joke about how there’s a holiday for everything, and some of the holidays are so bizarre, like there’s one called Crush a Can day dedicated to crushing cans. So for this project I wanted to make a chrome extension that displays the weird holidays of that day in the new tab everytime you open a tab. I chose chrome extension because I think its best suited for my purpose. I don’t think people would remember to open up an app or navigate to a website everyday to see what holiday it is, but people are using the internet all the time, and thus opening up new tabs often. Therefore it would be easy to access often as well.

I encountered some problems

- I was not too familiar with API’s, what they’re used for, how to make a request, and using keys.


- I was trying to find an API that gives me all the weird holidays, like World Mosquito Day, and their date. I couldn’t find one, so I settled with one that gave me just the normal holidays. It wasn’t working when I tried to access the data, and I spent days reading about requesting and accessing json data. It looked like I was doing everything right, so in the end I just started using this other API, which was very limiting, because the free version only offered official Romanian holidays :( it worked, but the free version doesn’t last very long and the api key needs to renewed after every couple of hours.

Accessing data

- I had absolutely zero experience with this and had no idea how to go about getting data from the URL and displaying it on the webpage. Google helped me a lot, but i also had to know what I didn’t know in order to utilize google.

What I learned

- From using web API's, I now know what a query and a query builder is XD. I also now know that for web based api’s, the url link itself is actually like a api request, with it’s ? and = signs, which makes sense and is kinda cool.


- I now know what json is and how easy it is to use and access its data


- I learned about using XMLHttpRequests and javascript to display json data taken from a URL on my webpage

- Of course, now I know how to make a chrome extension
